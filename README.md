# PerfilPro

**PerfilPro** é um sistema web desenvolvido com o objetivo de cadastrar, armazenar e gerenciar perfis profissionais voltados para entrevistas de emprego. A plataforma é voltada para empresas que desejam avaliar e selecionar candidatos com base em critérios previamente definidos.

## 💼 Objetivo

Criar uma solução parecida com o LinkedIn, porém mais focada no processo de recrutamento, permitindo que candidatos cadastrem seus dados e que recrutadores possam visualizar e filtrar perfis adequados para as vagas disponíveis.

## 🏫 Projeto acadêmico

Este projeto está sendo desenvolvido como parte do curso de **Análise e Desenvolvimento de Sistemas** da **Unialfa**.

## 🧑‍💻 Funcionalidades previstas

- Cadastro de usuário (candidato)
- Login e autenticação
- Formulário de perfil com informações como:
  - Experiência profissional
  - Formação acadêmica
  - Habilidades técnicas e comportamentais
  - Pretensão salarial
  - Disponibilidade de horário
- Banco de dados relacional em **PostgreSQL**
- Painel administrativo para visualizar e filtrar perfis

## 🛠️ Tecnologias utilizadas

- PostgreSQL (estrutura inicial do banco de dados)
- Git e GitHub para versionamento
- Outras tecnologias serão adicionadas futuramente
